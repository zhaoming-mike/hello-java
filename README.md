# hello-java

这是一个基础项目，用来编写一些基础的 java 类库。

# Checkout

`git clone https://github.com/zhaoming-mike/hello-java.git`

# Run

`mvn clean install exec:java -Dexec.args="hi 9999"`

# Test

[访问测试](http://localhost:9999/hi?github.com=http://github.com "access github.com")
